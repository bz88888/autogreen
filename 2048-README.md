# 2048 游戏

一个基于纯 HTML、CSS 和 JavaScript 实现的经典 2048 游戏。

## 功能特性

### 游戏功能
- ✅ 4x4 网格游戏板
- ✅ 初始随机生成两个方块（值为 2 或 4）
- ✅ 支持键盘方向键控制（上、下、左、右）
- ✅ 相同数字方块自动合并
- ✅ 每次移动后随机生成新方块
- ✅ 游戏结束检测

### UI 设计
- ✅ 清晰的 4x4 网格布局
- ✅ 实时显示当前分数和最高分
- ✅ 不同数字使用不同颜色区分：
  - 2: 浅米色 (#eee4da)
  - 4: 米色 (#ede0c8)
  - 8: 橙色 (#f2b179)
  - 16: 深橙色 (#f59563)
  - 32: 红橙色 (#f67c5f)
  - 64: 深红色 (#f65e3b)
  - 128: 金黄色 (#edcf72)
  - 256: 亮金色 (#edcc61)
  - 512: 深金色 (#edc850)
  - 1024: 深黄色 (#edc53f)
  - 2048: 金色 (#edc22e)
  - 更大值: 黑色 (#3c3a32)
- ✅ 方块出现和合并动画效果
- ✅ 新游戏按钮
- ✅ 响应式设计，适配移动设备
- ✅ 触摸滑动支持（移动端）

### 技术实现
- ✅ 纯 HTML、CSS、JavaScript（无框架依赖）
- ✅ 面向对象的游戏逻辑设计
- ✅ LocalStorage 保存最高分
- ✅ 流畅的动画过渡效果
- ✅ 移动端触摸滑动操作

## 如何使用

### 方法 1：直接打开
1. 下载所有文件（index.html、style.css、game.js）
2. 双击 `index.html` 文件在浏览器中打开
3. 开始游戏！

### 方法 2：使用本地服务器
```bash
# 使用 Python 3
python3 -m http.server 8000

# 或使用 Node.js (需要安装 http-server)
npx http-server -p 8000
```

然后在浏览器中访问 `http://localhost:8000`

## 游戏规则

1. 使用键盘方向键（↑ ↓ ← →）移动方块
2. 移动端可以使用手指滑动操作
3. 相同数字的方块碰撞后会合并成它们的和
4. 每次移动后会在空白位置随机生成一个新方块（2 或 4）
5. 当无法继续移动时游戏结束
6. 目标是创造出 2048 方块！

## 游戏控制

- **键盘方向键**: 移动方块
- **新游戏按钮**: 重新开始游戏
- **触摸滑动**: 移动端滑动屏幕移动方块

## 技术细节

### 核心类: Game2048
- `initGrid()`: 初始化游戏网格
- `addRandomTile()`: 添加随机方块
- `move(direction)`: 处理方块移动
- `mergeLine(line)`: 合并相同数字的方块
- `isGameOver()`: 检查游戏是否结束
- `render()`: 渲染游戏界面
- `updateDisplay()`: 更新分数显示

### 数据存储
使用 LocalStorage 保存最高分，即使关闭浏览器也会保留。
